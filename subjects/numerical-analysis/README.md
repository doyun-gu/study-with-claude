## Core Topics

1. **Root Finding** — Bisection, Newton-Raphson, Secant method, fixed-point iteration
2. **Interpolation** — Lagrange, Newton divided differences, splines
3. **Numerical Integration** — Trapezoidal rule, Simpson's rule, Gaussian quadrature
4. **Linear Systems** — Gaussian elimination, LU decomposition, iterative methods
5. **Eigenvalue Problems** — Power method, QR algorithm
6. **ODEs** — Euler's method, Runge-Kutta, stability analysis
7. **Error Analysis** — Floating-point arithmetic, truncation error, round-off error
